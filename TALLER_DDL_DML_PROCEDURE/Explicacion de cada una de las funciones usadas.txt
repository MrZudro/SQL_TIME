upper() función que permite volver mayúsculas
lower() función que permite volver minúsculas

substring()función que permite crear subcadenas a partir de una mas grande, esta recibe una cadena, el índice donde se crea y cuantos caracteres apartir de allí tomara para crear la subcadena.
